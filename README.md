# Introduction
A discord bot that was created for my discord server.

## Features
- Ticket Creation
- Ticket Logging
- Ticket Transcript
- Staff Commands
- Database Hookup using mySQL

## Dependencies
- Config.json
- DiscordJS
- Dotenv
- Jsdom
- mySQL12
- Sequelize
- Sleep
